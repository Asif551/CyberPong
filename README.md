# Neon Pulse Pong

A polished local two-player Pong-style game built with Python and Pygame.

## Run

```powershell
pip install -r requirements.txt
python main.py
```

## Controls

- Player 1: `W` / `S`
- Player 2: `Up` / `Down`
- Pause: `Esc`
- Menu confirm: `Enter` / mouse click
- Text entry: type names, `Tab` switches fields, `Enter` starts

The game creates procedural sound effects and synthwave loop assets in `assets/generated/` the first time it runs.
